## About Me

I'm **Matin Shahabadi**  
Also known as **متین شاه‌آبادی / متین شاه آبادی**

I'm a software developer based in Iran, focused on turning ideas into reliable, understandable, and useful systems. My work is mostly centered around **Python**, **Django**, **backend engineering**, automation, tooling, and clean web development.

I care about:

- Building software that is simple to use and easy to maintain
- Writing readable code with clear structure
- Improving developer workflows and project tooling
- Creating stable, documented, and practical solutions
- Learning modern backend architecture and scalable system design

Currently, I'm strengthening my skills in **Django**, **REST APIs**, backend architecture, deployment workflows, and production-ready web development.
